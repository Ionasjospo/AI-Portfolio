What does this mean?
This is the classification report and confusion matrix of an LDA model.

Classification Report:

  • Precision: Measures the accuracy of positive predictions. For example, precision for Basketball is 0.333, meaning the model was correct 33.3% of the times it predicted Basketball.
  • Recall (Sensitivity or True Positive Rate): Measures the model's ability to find all positive instances. It is the proportion of true positives over the total real positives. Example: Recall for Football is 0.684, meaning the model correctly identified 68.4% of Football examples.
  • F1-Score: The harmonic mean of Precision and Recall. It is useful for finding a balance between both, especially when a class is imbalanced. Example: The F1-Score for Rugby is 0.441, indicating a balance between precision and recall for this class.
  • Support: Number of real instances of each class in the test dataset. Example: Support for Football is 38, indicating there are 38 real examples of Football.
  • Accuracy: The proportion of correct predictions out of the total examples. In this case, the model has an overall accuracy of 40.5%.
  • Macro Avg: Unweighted average of Precision, Recall, and F1-Score across all classes, without considering the number of instances per class.
  • Weighted Avg: Weighted average of Precision, Recall, and F1-Score, considering the number of instances in each class.

Confusion Matrix:
The confusion matrix displays how the model's predictions are distributed compared to the actual values. Each row represents the real values and each column represents the model's predictions.

Example:
In the first row [3 7 12 6]:

  • The model correctly predicted Basketball 3 times.
  • It predicted Rugby instead of Basketball 7 times.
  • It predicted Volleyball instead of Basketball 12 times.
  • It predicted Football instead of Basketball 6 times.
